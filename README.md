## How to start

1. first clone the repo with the following command
    ```shell
      git clone https://github.com/skjghisole/DinoByte-.git
      cd Dinobyte-
    ```
2. install Packages
    ```shell
      (NPM) npm i
      (yarn) yarn install
    ```
3. Build and start your server!
    ```shell
      gatsby build
      gatsby serve
    ```
    for hot reloading every changes run this
    ```shell
      gatsby develop
    ```
